# HealthCare Clinic 🏥

A full-stack web application for a medical clinic built with React (Vite) and PHP.

## Features
- Home page with image slider
- Medical Services section
- Appointment booking
- Login system
- Contact page

## Tech Stack
- **Frontend:** React, Vite, Bootstrap 5, React Router DOM
- **Backend:** PHP, MySQL (PDO)

## Getting Started

### Prerequisites
- Node.js
- XAMPP (Apache + MySQL)

### Installation

1. Clone the repo
```bash
   git clone https://github.com/mohamed-mounib/medical-clinic.git
```

2. Copy the project to XAMPP:
3. 3. Create the database:
   - Open `http://localhost/phpmyadmin`
   - Create a database named `medical_clinic`
   - Import `back-end/database_schema.sql`

4. Run the frontend:
```bash
   cd my-react-app
   npm install
   npm run dev
```

5. Start Apache and MySQL from XAMPP Control Panel

6. Open `http://localhost:5173`
